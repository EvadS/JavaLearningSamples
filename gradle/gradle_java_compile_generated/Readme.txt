простой демонстративный проект Gradle + Java

проект состоит из двух директорий с исходными кодами 

src/main  - здесь лежат исходники
src/second - здесь лежат промежуточные файлы

изначально проект предназначался для кодогенерации 
во второй директории предполагалось разместить сгенерированные файлы

task fatJarfatJar предназначен для компиляции файлов из второй директории 
будет создана директория 'out/generated' где будет размещатся результирующие *classes

запуск 
 из build/libs/
java -jar generated_jar.jar
